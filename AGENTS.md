# Project Context: Motor Solver

## 1. Technology Stack
- **Python 3.12+**
- **Geometry**: Open CASCADE via OCP (Python bindings)

## 2. Coding Standards
- **Language**: Strict English for all code, docs, and commits.
- **Typing**: Mandatory Python type hints; avoid `Any`.
- **Docs**: Google-style docstrings for all functions/classes.
- **Style**: Concise, elegant, and Pythonic (PEP 8).
- **Comments**: Brief inline notes for complex logic/math only.
- **Performance**: Use NumPy vectorization; avoid loops for math.
